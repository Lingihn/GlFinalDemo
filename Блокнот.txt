Задание
В соревновании по велоспорту участвуют по 4 гонщика из каждой страны. О каждом гонщике известны следующие данные: 
номер, фамилия, имя, рейтинг. Соревнование проходит в два круга. Первый круг состоит из N этапов. Место страны на этапе 
определяется по времени худшего  (занявшего четвертое место) гонщика этой страны на рассматриваемом этапе. 
Место страны в итоговом протоколе первого круга определяется по наименьшей сумме мест, занятых страной на этапах, а при равенстве 
этого показателя – по более высокому месту в итоговом протоколе (в личном зачёте) лучших гонщиков конкурирующих стран. 
Во второй круг 
проходит 4 страны имеющие лучший результат. Второй круг проходит в два этапа. Место страны на этапе определяется по времени лучшего 
(занявшего первое место) гонщика этой страны на рассматриваемом этапе. Место страны в итоговом протоколе второго круга определяется 
по наименьшей сумме мест, занятых страной на этапах, а при равенстве этого показателя – по средне арифметическому значению рейтинга 
всех гонщиков страны.
 



1. Соревнование по велоспорту. В 2 этапа.
2. Количество стран не оговорено.
3. От каждой страны участвуют по 4 гонщика.
4. Каждый гонщик имеет поля - НОМЕР, ФАМИЛИЯ, ИМЯ, РЕЙТИНГ.
5. Первый круг - N этапов.
6. Последний гонщик от страны определяет место на этапе.
7. Победитель круга - тот, у кого наименьшее число мест.
8. Если между двумя странами сумма мест равна - по личному зачету между странами.

После проведения круга - Итоговый протокол по гонщикам и по странам.
После вывода - подтверждение пользователем для старта второго круга.

9. 4 Лучших страны попадают во второй этап.
10. Второй круг в 2 этапа.
11. Страна - победитель этапа - гонщик с наименьшим временем. В обоих этапах 
12. Победитель круга - тот, у кого наименьшее число мест.
13. Если между двумя странами сумма мест равна - среднее арифметическое значение мест всех гонщиков страны.

Итоговый протокол по второму кругу.
Сообщение о стране победителе.

Данные для ввода:
1. Количество стран участников.
2. Количество этапов.

Данные для генерации:
1. Время каждого гонщика в интервале.
2. Фамилия и Имя гонщика. 

Данные для вывода:
1. Протокол 1 круга по гонщикам.
2. Протокол 1 круга по странам.
3. Протокол 2 круга по гонщикам.
4. Протокол 2 круга по странам.
5. Страна победитель 

Куда выводить:
1. Алертами и Промптами.

Данные для расчетов:
1. Накопительные:
1.1 Для общего зачета - количество побед страны в этапах.


Функции. 
Для реализации гонки.
1. Функция для вычисления результата гонщика для каждого элемента 
2. Функция,которая разделит всех гонщиков на страны.
3. Функция, котрая найдет худший результат для каждой страны.
4. Функция, которая для каждой страны добавит индекс.
5. Функция, которая найдет наименьший индекс среди всех стран.
6. Функция, если индексы равны - ищем лучший результат. 

Для определения победителя в круге.

Примечание: После проведения первого круга соревнований, должен быть выведен итоговый протокол по гонщикам и по странам. 
Для проведения второго круга требуется подтверждение пользователя. Затем выводится итоговый протокол по второму кругу и 
отдельно сообщение о стране-победителе соревнований.



Критерии оценки задания:

1) Имена переменных (классов и функций) позволяют понять какие данные в них хранятся.

2)    Классы и функции, используемые в программе, расположены в одном месте и не перемежаются 
со строками основной программы (например, сначала идут классы, затем функции, затем текст основной программы).

3)    Рациональное использование функций (процедур) (либо они предназначены для однотипной обработки данных, 
либо выполняют логически завершённую часть кода, например, случайная генерация исходных данных, ввод исходных данных с клавиатуры).

4)    Использование структур данных (классы, объекты, массивы).

5)    Соблюдаются дли требования по синтаксису программного кода (пробелы, отступы, длинные строки).

6)    Использование комментариев, в том числе к функциям.

7)    Наличие проверки на ввод корректных данных.

8)    Задача решена правильно (выводится результаты в нужном виде как указано в примечании)
